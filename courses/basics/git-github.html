Herramientas a usar

La herramientas que se usaron en el curso básico de react en platzi son:

  • React Developer Tools: Es una herramienta Open Source creada por Facebook y tiene instalación para chrome o firefox. Nos dejará ver el código de React inspeccionando elementos.
  • Editor de texto: Puedes usar cualquiera, en este curso se sugirió Visual Studio Code.

VS Code tiene muchos plugis utiles entre los cuales esta:
Prettier Es un plugin que hace que el código se vea bien sin importar cómo esta escrito. Tiene la función de que al darle save, le da formato al código.

Para activar la función de que al guardar el documento se haga el autoformateado, se debe habilitar la opción editor.formatOnSave:
  • Abril el Command Palette, control + shift + p (en windows)
  • Escribir Open user Settings
  • Buscar format on save y activarlo

Para deshabilitar prettier para ciertos lenguajes, se puede emplear la siguiente configuración, en el archivo setting.json de VS Code:
"prettier.disableLanguages" : ["html"].
