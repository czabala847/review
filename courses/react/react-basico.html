Curso básico de react js

React cumple su función como biblioteca ya que para utilizar su código se debe importar. También es un Framework aunque las convenciones de cómo debe ser organizado todo no son estrictas.
React es declarativo, lo que quiere decir que se le indica qué debe hacer pero no cómo debe hacerse, ahorrando de esta manera muchos pasos.
React está estructurado por componentes que son como pequeños bloques de lego que al ser unidos forman aplicaciones de React. Estos componentes pueden tener estilos, ser enlazados a eventos y sus estados pueden ser modificados. Con React también se tiene la ventaja de que será escrito una sola vez y podrá ser utilizado en aplicaciones web, móviles, entre otras.

Herramientas a usar

La herramientas que se usaron en el curso básico de react en platzi son:

  • React Developer Tools: Es una herramienta Open Source creada por Facebook y tiene instalación para chrome o firefox. Nos dejará ver el código de React inspeccionando elementos.
  • Editor de texto: Puedes usar cualquiera, en este curso se sugirió Visual Studio

VS Code tiene muchos plugis utiles entre los cuales esta:
Prettier Es un plugin que hace que el código se vea bien sin importar cómo esta escrito. Tiene la función de que al darle save, le da formato al código.

Para activar la función de que al guardar el documento se haga el autoformateado, se debe habilitar la opción editor.formatOnSave:

  • Abril el Command Palette, (control + shift + p) en windows
  • Escribir Open user Settings
  • Buscar format on save y activarlo

Para deshabilitar prettier para ciertos lenguajes, se puede emplear la siguiente configuración, en el archivo setting.json de VS Code:
"prettier.disableLanguages" : ["html"]

Create React App

Create React App app es una aplicación moderna que se usa desde una línea de comando. Antes de ella se configuraba todo el entorno manualmente lo cual tomaba mucho tiempo.

Instalar de forma global: 

                  npm install -g react-create-app
