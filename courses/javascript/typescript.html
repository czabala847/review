TypeScript

TypeScript es un superset de JavaScript que añade tipos a nuestras variables ayudando así a la detección de errores de forma temprana y mejorando el autocompletado. Los navegadores no entienden TypeScript así que hay que transpilar a JavaScript usa Parcel.

Curso platzi
  • Parcel

Transpilar código de TypeScript a JS

Instalar parcel en el proyecto a través de npm:


                npm install -D parcel-bundler

Hay que reemplazar en script del archivo package.json y configurar los HTML que queremos que parcel procese. 


                "scripts": {
                  "start": "parcel index.html ejercicios/index.html ejercicios/**/*.html"
                },

Se pueden añadir configuraciones, ejemplo decirle a parcel que transpile el código para el navegador que esta utilizando, que es una versión reciente de chrome. 


                //Browserlist: que navegadores queremos darle apoyo.
                "browserlist": [
                  "last 1 Chrome version"
                ]

Si antes estaba instalado liveServer debemos desistalarlo 


                npm rm live-server

Ahora se podrá correr el servidor de parcel, mediante npm start, si no es así se debe apagar el servidor y borrar las carpetas .cache y dist 


                rm -rf .cache dist
